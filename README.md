# Starship Sentinel
Starship game which makes heavy use of transformation buffers.

Press any key to move on from the splash screen.


Select a level 1-4

Control your game with:
Q to thrust
O and P to rotate
A to fire

Try to destroy all the enemy craft before your fuel or air run out. 

Don't crash into the enemy.

Land back at your base station to refuel if needed.

Press ESC to exit the game.
